## Upgrading Your Documentation Space

Follow these steps to upgrade to the latest version safely.

<Steps>
  <Step title="Backup Your Space" icon="database">

    Export your current documentation:

````bash
# Using CLI tool
roman-docs export --space-id YOUR_SPACE_ID --output backup.json
````

  </Step>
  <Step title="Update Configuration" icon="settings">

    <CodeGroup tabs="JSON,YAML">
````json
{
  "theme": {
    "primaryColor": "#3B82F6"
  },
  "features": {
    "collaboration": true
  }
}
````
````yaml
theme:
  primaryColor: "#3B82F6"
features:
  collaboration: true
````
    </CodeGroup>

  </Step>
  <Step title="Run Upgrade" icon="rocket">

    Deploy the update:

````bash
roman-docs upgrade --version 1.2.0 --space-id YOUR_SPACE_ID
````

  </Step>
</Steps>

<Expandable title="Detailed Migration Notes" default-open="false">

For v1.0.0 to v1.2.0:

1. Convert YAML configs to JSON using the provided CLI converter.
2. Update any custom CSS to match new theme variables like `{primaryColor}`.
3. Test collaboration features in a staging space first.

| Old Config | New Config |
|------------|------------|
| `theme.color: blue` | `{"theme": {"primaryColor": "#3B82F6"}}` |
| `search.enabled: true` | `{"features": {"search": true}}` |

If you encounter issues, contact support with your space ID.

</Expandable>
